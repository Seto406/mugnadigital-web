# Social Media Templates

This directory contains HTML/Tailwind CSS templates for social media posts (Instagram/Facebook/LinkedIn).

## How to use
1. Open the HTML files in a web browser.
2. The containers are scaled to 540x540px for viewing, but represent a 1:1 aspect ratio.
3. Take a screenshot of the container content.
4. Or, copy the code and adjust the `w-[540px] h-[540px]` classes to `w-[1080px] h-[1080px]` and use a tool like Puppeteer or a browser screenshot tool to export high-resolution images.

## Templates
- **Announcement (`post_1_announcement.html`):** For big news, launches, or updates.
- **Quote (`post_2_quote.html`):** For testimonials, inspiration, or team quotes.
- **Feature (`post_3_feature.html`):** To highlight specific features or services.
